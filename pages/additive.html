3D Print Job Process

  1. Read the content and look through the guides on this page.
  2. Ensure your file is in .3MF, .STL, or .OBJ format.
  3. Name your file using this format: (FirstLast_Method_Color.ext)
    e.g., JaneDoe_Filament_Blue.stl
  4. Click the "3D Print Submission" button below and complete the form, including uploading your file.
  5. You will receive a confirmation email with your Request ID.
  6. Our team will review your file to verify it is printable.
  7. If the file is not printable, we will email you with recommendations for fixes.
  8. When the file is ready, we will slice your model and email you a price estimate.
  9. Click the link in the estimate email to confirm the price. Printing will not proceed until you confirm.
  10. We will print your job using the most suitable 3D printer.
  11. Upon completion, you will receive an email notification with pickup instructions.
  12. Payment is required at the time of pickup via TigerCASH.

Model Checklist

  1. Did you submit your file with the correct name format?
  2. Has the model been broken down to reduce support material?
  3. Is it a valid Closed Solid Polysurface or closed mesh? (Type "What" into the command bar in Rhino to check validity)
  4. Is the model scaled to the correct size? .STL and .OBJ MUST be exported and scaled in metric units otherwise the scale will be wrong.
  5. Does the scaled size fit within the build volume? (You can select your model and type "Bounding Box" to double-check the dimensions)
  6. Is your model at least 1/16" thick when scaled to fit on a printer bed?
  7. You can include many items in one file. We simply arrange it ourselves to fit on the printers.
  8. Ensure that the file contains ONLY what you want printed. Export selected geometry and save it as .3mf, if possible, to ensure accuracy. If it says a plugin failed, save the geometry you want printed as a new Rhino file and proceed to export from there.
  9. Send us ONE SINGLE file whenever possible. Multiple files as a last resort.
